# Mech It Easy (by Aakarantech)

Mech It Easy is a premium, future-ready platform dedicated to building innovation and science labs for schools and colleges. We focus on hands-on environments for science, engineering, and digital fabrication—from initial layout planning to everyday classroom operations.

## 🚀 Mission
To help educational institutions design learning spaces where ideas turn into experiments, prioritizing safety, learning outcomes, and teacher-friendly operations.

## 🛠️ Key Offerings

### 1. Mech It Easy Kits
Concept-driven hardware kits built for repeated classroom use.
- **Hand Crank Generator**: Mechanical to electrical energy conversion.
- **Pulley Kit**: Mechanical advantage and force distribution.
- **Lever Kit**: Interaction of force, load, and fulcrum.

### 2. Mech It Easy Labs
End-to-end digital fabrication and science lab setup solutions.
- **Layout Planning**: Bench placement and supervision lines.
- **Organization**: Color-coded storage and tool check-in systems.
- **SOPs**: Playbooks and routines for sustainable lab operations.

### 3. Digital Fabrication (3D Printing Services)
Classroom-safe workflows for modern manufacturing tools.
- **3D Printing**: Standardized queues and material guidelines.
- **Laser Cutting**: Project-ready templates and safety rules.
- **CNC Routing**: Prototyping workflows for foam, wood, and plastics.

### 4. Training & Capacity Building
Structured training modules for students, teachers, and instructors.
- **Student Courses**: Modular courses from idea to 3D model.
- **Teacher Support**: Curriculum integration and machine maintenance.

## 💻 Tech Stack
- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Icons**: Lucide (via React Icons)
- **Forms**: Formspree (Serverless submission)
- **Animations**: Intersection Observer for scroll-triggered transitions.

## 🏗️ Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
1. Navigate to the client directory:
   ```bash
   cd client
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🎨 Design Philosophy
- **Premium Aesthetics**: Clean gradients, glassmorphism (backdrop-blur), and smooth micro-animations.
- **Outcome-First**: Design elements that highlight educational value rather than just technology.
- **Calm Rollout**: Minimalist UI to prevent information overload for educators and students.
